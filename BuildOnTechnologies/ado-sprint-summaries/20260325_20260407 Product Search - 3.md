**State:** Done | **Priority:** 1 | **Tags:** product search; scripts

**Description:**

Feature Description:  
As a designer or sales counselor...  
As category structures change over time, designers (or sales people) can be unaware of the location of certain items. An important customer has asked  
us if we can implement a global product search to help them locate options easier.  
 
As a buyer...  
I may be interested to determine if a specific style of product exists in my builder program. I'd like an easy way to search and determine if that is the case. It would be good if the system would locate something once I identify it exists.  
 
 
As a Product Owner/Manager:  
I want to know when this feature is being used, and how often so that I can make better decisions about continuing to invest in it.  
 
--------------------------------------------------------------------------------------------------------- 
This work item covers usage of product search and the find functionality in option pricing modules. 
 
---------------------------------------------------------------------------------------------------------

**Acceptance Criteria:**

Given the feature flag for product search is on  
and the product Search Component is functional (though potentially not yet fully styled)  

 
Elevate Price Level in search results & display on "card"  
Currently, the state of the search component is such that the price level for a given "hit" is part of the area related meta data. We think we need to elevate the price level out of that meta data to the top level result fields, so that it be part of the aggregation. Meaning, that if I search for a tile called "Arctic White", I would get a single result for Flooring->Tile Price Level X and another result for Walls->Tile Price Level Y (different price level for this part in wall tile).  

 
Embed in Option Pricing Nav Menu  
the search component is embedded in the option pricing search for both product catalog and (regular) option pricing  
 Perform Search  
the user can perform a search that produces at least 1 result  
we expect the search results to NOT be fully styled
